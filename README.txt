ProteinLocations
================


A Database of Protein Locations, combined together from different sources. 


1. Scripts to extract Protein information from databases like UniRef, HPRD, SUBA, etc. 
2. The downloaded information from the above databases is in XML format.
3. Used Python ElementTree Parser to parse the XML file.
4. Extracted required protein information using RegEx python.
5. Inserted the fields in MySQL database.
6. Developed a command line UI to query this protein locations database.
